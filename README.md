# 03-textures-and-camera-movements

Learning exercise for **Three.js + TypeScript**, focused on fixture load and camera movement.

## Goal

Recreate a walking path with stars in the sky and some obstacle. The user will be able to move by using the keyboard.

## Stack

- TypeScript
- Three.js
- Parcel (bundler/dev server)

## Scripts

- `yarn start` — run the dev server
- `yarn build` — production build

## Deployment

The project will be published on **GitHub Pages**.
